## 1. Overview

**ExpoCore** هي منصة متكاملة لإدارة المعارض تستهدف **المستثمرين (أصحاب الأجنحة)**، وتُعلن المعارض والفعاليات من خلال **نسخة الإدارة**، فيما يتصفّح المستثمر المعارض المتاحة، ويحجز أجنحة، ويشارك في فعاليات، وينشئ فعاليات خاصة بجناحه، ويتابع أداء جناحه عبر تقارير تحليلية تفصيلية.

### 1.1 Problem Statement
تعاني المعارض من فوضى في إدارة الأجنحة والتواصل، نتيجة الاعتماد على البريد الإلكتروني وجداول Excel — مما يُسبب تأخيرات وأخطاء في تخصيص الأجنحة وصعوبة متابعة حالة كل مستثمر.

### 1.2 Solution
منصة موحّدة للمستثمر تتيح:
- **تصفّح المعارض** المُعلَنة من قِبَل الإدارة
- **حجز الأجنحة** في المعارض المختارة
- **حفظ المفضلة** للمعارض والفعاليات والأجنحة
- **إدارة الحضور** بالكامل: الملف التعريفي، الحملات، الفعاليات، التحليلات، التقارير، والتواصل
- **عرض وتنزيل التقارير التحليلية** الخاصة بأداء الجناح

### 1.3 Tech Stack

| Layer | Technology |
|---|---|
| Mobile Frontend | Flutter (Dart) |
| State Management | GetX |
| Backend API | Laravel REST API |
| Auth | JWT / OAuth via Laravel |
| Local Storage | SharedPreferences + FlutterSecureStorage |
| HTTP Client | Dio |
| Charts | fl_chart |
| Push Notifications | Firebase Cloud Messaging (FCM) |
| 3D Map | Flutter WebView + Three.js |
| PDF Generation | pdf (dart) + printing |
| File Download | path_provider + open_file |

- **Scale:**
  - `headlineLarge`: 28sp / Bold
  - `headlineMedium`: 22sp / SemiBold
  - `titleLarge`: 18sp / SemiBold
  - `titleMedium`: 16sp / Medium
  - `bodyLarge`: 14sp / Regular
  - `bodySmall`: 12sp / Regular
  - `labelSmall`: 10sp / Medium

### 2.6 Spacing & Radius
- Base unit: `8dp`
- Card radius: `16dp`
- Button radius: `12dp`
- Input radius: `10dp`
- Small chip radius: `8dp`
- Favorite button radius: `50%` (circular)

### 2.7 Shared Components

#### `CustomButton`
- Full-width gradient button (darkPrimary → darkAccent)
- Height: `52dp`
- Loading state: centered CircularProgressIndicator (white)
- Disabled state: 50% opacity

#### `CustomTextField`
- Dark: `darkCard` background / Light: `lightCard` background
- Prefix/suffix icon support
- Validation error state (error color border + message)
- Password visibility toggle

#### `CustomAppBar`
- Back button (leading)
- Centered title (Cairo SemiBold)
- Optional action icons (trailing) — e.g. heart (favorites), share

#### `BottomNavBar`
5 tabs: الرئيسية · المعارض · أجنحتي · المفضلة · المزيد

- Active: gradient icon (darkPrimary → darkSecondary) + label
- Inactive: grey icon

#### `FavoriteButton` ★ جديد
- أيقونة قلب قابلة للضغط
- حالة نشطة: gradient fill (darkSecondary → darkAccent)
- حالة غير نشطة: outlined في اللون الثانوي
- Animation: `ScaleTransition` + `BounceIn` عند الضغط
- يمكن وضعها على أي بطاقة (معرض، فعالية، جناح)

#### `GlassContainer`
- Background: surface color with `0.15` opacity overlay
- `BackdropFilter` blur: `10`
- Border: `1dp` semi-transparent border (darkPink at 30% opacity)

#### `GradientBackground`
- Radial/linear gradient using brand colors
- Subtle animated particle effect on splash/onboarding

#### `AnimatedCard`
- `FadeInUp` entry animation (300ms)
- `ScaleTransition` on tap

#### `ReportCard` ★ جديد
- بطاقة عرض التقارير مع preview thumbnail
- زر "عرض" + زر "تنزيل" (PDF/Excel)
- Progress indicator أثناء التنزيل
- Date range display

#### `LoadingWidget` / `EmptyWidget`
- Lottie animation files
- Localized messages

## 8. State Management (GetX)

### Favorites Controller

```dart
class FavoritesController extends GetxController {
  final statusRequest        = StatusRequest.none.obs;
  final favoriteExhibitions  = <ExhibitionModel>[].obs;
  final favoriteEvents       = <EventModel>[].obs;
  final favoriteBooths       = <BoothModel>[].obs;
  final selectedTab          = 0.obs;

  @override
  void onInit() {
    fetchAllFavorites();
    super.onInit();
  }

  bool isExhibitionFavorited(int id) =>
      favoriteExhibitions.any((e) => e.id == id);

  bool isEventFavorited(int id) =>
      favoriteEvents.any((e) => e.id == id);

  bool isBoothFavorited(int id) =>
      favoriteBooths.any((e) => e.id == id);

  Future<void> toggleFavoriteExhibition(ExhibitionModel exhibition) async {
    if (isExhibitionFavorited(exhibition.id)) {
      favoriteExhibitions.removeWhere((e) => e.id == exhibition.id);
      await crud.postData('${AppLink.investorFavorites}/exhibitions/${exhibition.id}', {});
    } else {
      favoriteExhibitions.add(exhibition);
      await crud.postData('${AppLink.investorFavorites}/exhibitions', {'id': exhibition.id});
    }
  }
  // نفس النمط لـ Events و Booths
}
```

### Reports Controller

```dart
class ReportsController extends GetxController {
  final statusRequest  = StatusRequest.none.obs;
  final reports        = <ReportModel>[].obs;
  final selectedReport = Rx<ReportModel?>(null);
  final selectedType   = 'all'.obs;
  final dateFrom       = Rx<DateTime?>(null);
  final dateTo         = Rx<DateTime?>(null);
  final isDownloading  = false.obs;
  final downloadProgress = 0.0.obs;

  Future<void> downloadReport(String reportId, {String format = 'pdf'}) async {
    isDownloading.value = true;
    try {
      final dir = await getApplicationDocumentsDirectory();
      final path = '${dir.path}/report_$reportId.$format';
      // Download via Dio with progress callback
      await dio.download(
        '${AppLink.investorReports}/$reportId/download?format=$format',
        path,
        onReceiveProgress: (received, total) {
          downloadProgress.value = received / total;
        },
      );
      await OpenFile.open(path);
    } finally {
      isDownloading.value = false;
    }
  }
}
```

### نمط الـ CRUD (`core/class/crud.dart`)

```dart
class Crud {
  Dio dio = Dio();

  Future<Map> postData(String url, Map data) async {
    try {
      Response response = await dio.post(url,
        data: data,
        options: Options(headers: {
          "Authorization": "Bearer ${Get.find<Services>().token}"
        }),
      );
      return {"status": true, "data": response.data};
    } on DioException catch (e) {
      return {"status": false, "code": e.response?.statusCode};
    }
  }
}
```
